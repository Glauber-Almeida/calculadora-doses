Ícones adicionados para melhorar a aparência do atalho na tela inicial do iPhone.

Arquivos principais:
- apple-touch-icon.png: ícone usado pelo Safari/iOS ao adicionar à Tela de Início.
- apple-touch-icon-precomposed.png: fallback para compatibilidade.
- favicon-32.png: favicon pequeno para navegadores.
- site.webmanifest: metadados básicos do aplicativo.
